# Examen POO JavaScript

## Descripción
Proyecto realizado en JavaScript utilizando Programación Orientada a Objetos (POO) en Node.js.

El programa permite trabajar con empleados, desarrolladores y gerentes, aplicando conceptos de:
- Clases
- Herencia
- Encapsulamiento
- Getters y Setters
- Métodos

Clases Implementadas

Empleado
Clase base que contiene:
- nombre
- cédula
- salario base

Desarrollador
Hereda de la clase Empleado y agrega:
- lenguaje de programación
- cálculo de salario neto

Gerente
Hereda de la clase Empleado y agrega:
- departamento
- aumento salarial

Tecnologías Utilizadas
- JavaScript
- Node.js
- Git y GitHub

Cómo Ejecutar el Proyecto

Abrir la terminal en la carpeta del proyecto y ejecutar:

bash
node main.js
